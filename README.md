# File-Fridge
The task for the speed of light

Суть "File Fridge" заключается в том, чтобы сортировать файлы по времени последней их редакции.
При запуске кода на рабочем столе ПК устанавливается папка File Frige и 3 подпапка: hotbox, fridge, freezer.
Пользователь перетаскивает любой документ в папку File Fridge и через 60 секунд данный документ переносится в одну из подпапок.
(Редактирование было менее 3 дней назад - hotbox; более 3 дней, но менее 14 - fridge более 14 дней назад - freezer)
Раз в 24 часа происходит сортировка внутри трех подпапок. И все документы перемещаются в необходимый раздел.
Данной "холодильник" не сортирует папки и системные файлы. 
Если добавляется файл с названием ,которое уже существует, то "холодильник" добавляется суффиксы от дублирования. (_1, _2 и тд.)
